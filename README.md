# 🚀 AI Career Assistant

An AI-powered web application that helps users:

- 🎯 Predict suitable career paths based on skills
- 📊 Analyze skill gaps for target roles
- 📄 Upload resume and get automated insights

## 🛠️ Tech Stack
- Python
- FastAPI
- Streamlit
- Scikit-learn

## 💡 Features
- Career prediction using ML
- Skill gap analysis
- Resume-based skill extraction

## ▶️ Run Locally

Backend: python -m uvicorn backend.main:app --reload


Frontend: python -m streamlit run frontend/app.py
